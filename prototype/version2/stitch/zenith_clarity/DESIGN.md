# Design System Theme

This document outlines the core aesthetic and functional properties of our design system.

## Color Palette

Our color palette is built around a "light" color mode, ensuring a bright and accessible user experience. The primary brand color is a deep indigo (`#1A237E`), which serves as the anchor for interactive elements and key brand messaging. It is complemented by a vibrant green secondary color (`#2E7D32`) for supporting actions and elements, and a striking red tertiary color (`#D32F2F`) for accents or alerts. A clean, very light blue (`#F8F9FF`) is used as our neutral base for backgrounds and surfaces.

## Typography

We utilize "publicSans" across our typography system to maintain consistency and legibility.
- **Headlines:** `publicSans`
- **Body Text:** `publicSans`
- **Labels:** `publicSans`

## Shape and Spacing

Our UI elements feature a "moderate" level of roundedness (value `2`), providing a balanced and modern appearance. Spacing is set to "normal" (value `2`), which offers a comfortable amount of whitespace between and within elements, contributing to a clean and uncluttered layout.
